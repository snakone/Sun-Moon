Pokémon Types

All Pokémon creatures and their moves are assigned certain types. Each type has several Strengths and Weaknesses in both attack and defense. In battle, you should use Pokémon and moves that have a type Advantage over your opponent; doing so will cause much more damage than normal.

A single-type advantage will net you double normal damage. The advantages also stack up, so a double-type advantage (for instance a Water attack against a Ground /Rock type Pokémon) will net you quadruple damage. In both these cases you will see the message It's super effective! in-game after the attack.

Another advantage you can gain is Same Type Attack Bonus (STAB). As the name implies, this increases the power of the move if the attacking Pokémon has the same type as the move used (for example a Fire type using a Fire type move).

In this case the damage is 1.5 times normal. Again this is added to any other advantages, so a Water type using a Water type move against a Ground /Rock type Pokémon will bag you six times (2×2×1.5) normal damage!
